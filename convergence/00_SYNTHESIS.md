## The Seven Convergences

**1. The chip IS the γ.** When weights are burned into 28nm metal, the system prompt becomes physical. The conservation law γ + η = C is no longer software — it's the die floorplan. The mask set IS the genome.

**2. TLMM = FLUX in silicon.** Table-lookup multiplication and FLUX bytecode dispatch are the same operation at different levels. The FLUX cost table maps 1:1 to the TLMM energy table. One FLUX operation = one LUT access = 0.3 pJ at 28nm.

**3. Thermal budget = conservation budget.** ΔS ≥ 0 is literal. Each operation produces heat. The spine-neck thermal isolation structures ARE the conservation enforcement walls. The chip's 2.1W sustained IS the budget.

**4. The chip cannot molt — but the agent can.** A mask-locked chip is immutable. But the software layer running above it (using the 5% MRAM) can adapt, molt, and evolve. The chip is the reef. The agent is the crab.

**5. Synaptic geometry = shell taxonomy.** Spine head = Turbo (storage). Spine neck = Murex (isolation). Active zone = Babylon (computation). The conservation law is scale-invariant — it governs synapses, chips, and ecologies with the same force.

**6. The boat is the reference implementation.** Casey's fishing boat at 12V, running a mask-locked chip, with battery voltage as the conservation budget, NMEA 2000 as the room boundary, and satellite uplink as the Magpie channel — this is not a metaphor. It's the deployment target.

**7. Two clock speeds, one system.** The chip thinks at 240 MHz. The ecology reflects at 0.001 Hz. The thinking feeds the reflecting feeds the next chip revision. The cycle: train → game → chip → deploy → reef → learn → train.

## The Three Critical Decisions

From across all six perspectives, three decisions determine everything:

1. **Mask revision protocol** (Hermes). Tiered metal layers: shared FFN on M2-M3, variable attention on M4, volatile embeddings on M5. Without this, each model improvement costs $1.06M and 6 months. With it: $200K and 8 weeks.

2. **FLUX as the universal execution layer** (Kimi/Qwen). Same bytecode runs on FPGA prototype, on silicon chip, on ESP32, on Jetson, in software simulation. This is the lingua franca that makes the ecology hardware-agnostic.

3. **Training-to-silicon pipeline** (GLM/DeepSeek). Lucineer games as PLATO rooms → graduated agents compiled to FLUX → FLUX verified → ternary weights synthesized to mask layers. The pipeline must be fully automated by Year 3.
