## The Arc

The corpus went: metaphor → perspectives → arguments → futures → synthesis → reverse-engineering → **math and metal**.

Each step tightened the focus. The perspectives were expansive (800-1,200 words each, creative). The reverse-engineering was analytical (1,500-2,000 words each, structural). The math-and-metal is *formal* — equations, specs, code.

The arc is: **what does this feel like?** → **what does this mean?** → **what does this compute?** → **how do we build it?**
